# Rubendecoled - Blog en GitHub Pages

Este es el repositorio que contiene el blog estático optimizado para SEO, diseñado para conectarse con tu página web principal en Wix.

## Cómo publicar este sitio en GitHub Pages

Sigue estos pasos detallados para que tu blog esté visible en internet de forma gratuita:

### Paso 1: Crear el repositorio en GitHub
1. Inicia sesión en tu cuenta de GitHub (crea una si no tienes).
2. Haz clic en el botón verde **"New"** (o "+" en la esquina superior derecha -> "New repository").
3. En **Repository name**, escribe exactamente: `rubendecoled.github.io`
   *(Nota: si tu usuario en GitHub no es "rubendecoled", pon `tu-usuario.github.io` o cualquier nombre de proyecto, pero la URL final variará. Lo ideal es nombrarlo `rubendecoled.github.io`).*
4. Asegúrate de que esté marcado como **Public** (Público).
5. Haz clic en **Create repository**.

### Paso 2: Subir los archivos
1. En la página del repositorio recién creado, haz clic en el enlace que dice **"uploading an existing file"**.
2. Arrastra y suelta todos los archivos y carpetas de este proyecto (`index.html`, `styles.css`, y la carpeta `blog` entera) en la ventana del navegador.
3. Baja hasta el final de la página y haz clic en el botón verde **"Commit changes"**.

### Paso 3: Activar GitHub Pages
1. Ve a la pestaña **"Settings"** (Configuración) de tu repositorio.
2. En el menú lateral izquierdo, haz clic en **"Pages"** (bajo la sección "Code and automation").
3. En el apartado **Source** (o "Build and deployment" -> "Branch"), selecciona la rama `main` (o `master`) en el menú desplegable y asegúrate de que la carpeta sea `/ (root)`.
4. Haz clic en **Save**.
5. Espera un par de minutos. En la parte superior de esa misma página aparecerá un mensaje verde que dirá: *"Your site is live at https://rubendecoled.github.io/"*.

### Paso 4: Probar y Enlazar desde Wix
Una vez que el sitio esté activo, abre tu navegador y visita: `https://rubendecoled.github.io`

Para conectar este blog SEO con tu página de Wix, ve al editor de Wix y añade un enlace, botón o menú con la siguiente información:

*   **Texto sugerido para el botón/enlace:** `Lee nuestra guía para elegir servicio de coctelería para eventos`
*   **URL sugerida (Enlace web):** `https://rubendecoled.github.io/blog/como-elegir-servicio-cocteleria-evento.html`

¡Listo! Todo el tráfico que consigas desde Google a tu blog será redirigido hacia tu Wix a través de los botones internos del artículo.
